# SEEIK Dashboard

SEEIK 키오스크 운영 현황을 빠르게 확인하기 위한 내부 대시보드 프로젝트입니다.

---

## 프로젝트 소개

`seeik-dashboard`는 운영 데이터를 기반으로 다음을 한 화면에서 확인할 수 있도록 구성된 웹 대시보드입니다.

- 매장별 매출 추이 파악
- 일/월 단위 운영 지표 모니터링
- 트랜잭션 상세 조회 및 이상 징후 확인

상단 메뉴를 통해 **매출요약 / 일보 / 트랜잭션** 화면으로 이동하며,
매장 기준 비교가 가능하도록 설계되어 있습니다.

---

## 주요 기능

### 1) 매출 요약
- 오늘 매출/객수 요약 카드
- 컨셉(상세유형)별 구성비 시각화
- 기간 조건 기반 집계 조회
- 기간별 총 매출/총 주문 건수 요약

### 2) 일보(Daily Report)
- 연/월 기준 일별 매출 집계
- 월 진행률 기반 추정 월매출 확인
- 전월/전년 동기 비교 지표
- 월별 추이 차트
- 인쇄용 출력 화면 제공

### 3) 트랜잭션
- 기간별 상세 주문 내역 조회
- 매장 다중 선택 필터
- 오류 의심건 필터링
- 연속 발생 패턴 기반 이상 징후 표시
